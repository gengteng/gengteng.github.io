异或运算

发表于 2021-08-29 更新于 2025-01-24 阅读次数：

运算特性

  1. 两个数进行异或运算，二进制位相同为 0，不同为 1；
  2. 满足交换律和结合律；
  3. 异或运算可以认为是无进位相加（二进制）；
  4. 任何数与 0 异或仍得这个数，即0 ^ N = N；
  5. 任何数与自身异或得 0，即 N ^ N = N。
